# FentixTech Task 04 - Instagram UI

A full-page layout UI replica of Instagram, built with modern web development practices. This project demonstrates responsive design and modern aesthetics by recreating the Feed, Explore, and Profile pages of Instagram.

## Features

- **Feed Page (`feed.html`)**: Displays stories, posts with images, likes, and comments.
- **Explore Page (`explore.html`)**: Includes a search bar, categorization tags, and a dynamically styled image grid.
- **Profile Page (`profile.html`)**: Shows user statistics, bio, highlights, and a grid of the user's posts.
- **Full-Screen Layout**: Implemented as a full desktop web page, moving beyond basic mobile mockups.
- **Custom Profile Picture**: Integrated user-specific images seamlessly into the UI.

## Technologies Used

- **HTML5**: For semantic structuring of the application.
- **CSS3**: For advanced styling, flexbox layouts, and custom gradients.
- **Phosphor Icons**: For clean and modern vector icons.

## How to Run

Since this is a static web application, no server setup or build process is required.
Simply open any of the HTML files in your preferred web browser to view the project:

```bash
# Open in your browser, for example:
start instagram-ui/profile.html
```

## Folder Structure

- `instagram-ui/`
  - `css/`
    - `style.css` - Contains all styles for the application.
  - `feed.html` - The Instagram home feed.
  - `explore.html` - The explore section.
  - `profile.html` - The user profile page.
  - `1.jpeg` - The user's profile picture.
